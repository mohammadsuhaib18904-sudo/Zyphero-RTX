Privacy Policy for Zyphero-RTX

Effective Date: June 2026

Zyphero-RTX is a local-first, privacy-centric Integrated Development Environment (IDE) built by Mohammad Suhaib.

1. Data Collection

We collect zero personal data, source code, or telemetry. All AI processing happens either entirely locally on your machine via the bundled neural engine, or securely via the API providers you explicitly configure.

2. API Keys

Any premium API keys (e.g., Gemini, Groq, OpenRouter) entered into the application are encrypted and stored locally in your operating system's native secure Keychain. We do not have access to your keys.

3. Contact

For inquiries regarding this privacy policy, please contact the developer through GitHub Issues at: https://github.com/mohammadsuhaib18904-sudo/zyphero-privacy/issues
